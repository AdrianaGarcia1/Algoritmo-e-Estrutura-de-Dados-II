# Algoritmo-e-Estrutura-de-Dados-II(Fila e Pilha).
Autor: Nguinamau Dongala
Implementação das Estrutura de dados - Fila e Pilha

Este repositório é exclusivo para trabalho de Algoritmo e Estrutura de Dados, aplicando os conceitos abordados na aula 1, Pilha(Stack) e Fila(Queue) implementando o conceito de TAD(Tipos de Dados Abstracto) na linguagem java. 

Fila(Queue): Estrutura de dados que segue a ordem FIFO(First In, First Out).
Implementações:
-Inserir(enqueue)
-Remover(dequeue)
-Consulta do Primeiro elemento(front)
-verificar se a fila está vazia(isEmpty)


Pilha(Stack): Estrutura de dados com a ordem inversa à fila, ela seque o princípio de LIFO(Last In, First Out).
operações Implementadas:
Inserir(push)
Remover(pop)
Consulta do topo(peek)
Verificar se a pilha está vazia(isEmpty)
